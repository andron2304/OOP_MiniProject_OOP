# Бекліст продукту

## Ітерація 1: Фундамент і Вертикальний зріз
- [ ] Визначити сутність домену Track з властивостями (Title, Artist, Duration, Genre)
- [ ] Визначити сутність домену Playlist з підтримкою паттерну Composite
- [ ] Створити інтерфейс IPlaylistComponent (Track та Playlist його реалізують)
- [ ] Реалізувати in-memory IPlaylistRepository
- [ ] Реалізувати сервіс PlayerService на рівні Application
- [ ] Створити консольний UI для створення плейліста та управління треками
- [ ] Відобразити вміст плейліста з метаданими треків

## Ітерація 2: Збереження і розширені операції
- [ ] Реалізувати файл-базовано сховище (JSON серіалізація)
- [ ] Додати функціональність перемішування з варіантами алгоритмів
- [ ] Створити сервіс управління чергою
- [ ] Реалізувати пошук та фільтрування треків
- [ ] Додати видалення та змінення плейліста
- [ ] Створити міграцію даних між реалізаціями сховища

## Ітерація 3: Симуляція відтворення та аналітика
- [ ] Реалізувати імітацію відтворення аудіо (імітація тривалості відтворення)
- [ ] Додати відстеження історії відтворення
- [ ] Створити сервіс статистики (найбільш відтворювані, кількість відтворень)
- [ ] Реалізувати навігацію пропуску та попереднього треку
- [ ] Додати режим повторення (вимкнено, один, все)
- [ ] Створити дисплей прогресу для відтворення треку

## Ітерація 4: Інтеграція бази даних та покращення UI
- [ ] Реалізувати Entity Framework Core сховище
- [ ] Додати підтримку SQLite бази даних
- [ ] Створити розширене фільтрування (за тривалістю, датою додавання, кількістю відтворень)
- [ ] Реалізувати масове імпортування з файлової системи
- [ ] Додати збереження переваг та параметрів користувача
- [ ] Покращити консольний UI з пагінацією та сортуванням
- [ ] Додати обробку помилок та валідаційний рівень
